Wayra Web Backend is the server-side API for Wayra Nikkei restaurant. It exposes a set of HTTP endpoints that enable frontend clients to handle the full restaurant workflow: staff authentication, table reservations, order management, payment processing, and administrative reporting. The API is built with Express.js 5 and communicates with a MySQL database via a connection pool.

## Getting started

<Steps>
  <Step title="Clone and install">
    Clone the repository and install dependencies with npm.

    ```bash
    git clone https://github.com/carrioneilnacr-art/Wayra_Web_Backend.git
    cd Wayra_Web_Backend
    npm install
    ```
  </Step>
  <Step title="Configure your database">
    Create a `.env` file and set your MySQL connection string.

    ```bash .env
    DATABASE_URL=mysql://user:password@host:3306/wayra_db
    PORT=3000
    ```
  </Step>
  <Step title="Start the server">
    Run in development mode with auto-reload, or start for production.

    ```bash
    npm run dev   # development with nodemon
    npm start     # production
    ```
  </Step>
  <Step title="Make your first request">
    Call the login endpoint to verify the server is running.

    ```bash
    curl -X POST http://localhost:3000/api/login \
      -H "Content-Type: application/json" \
      -d '{"user": "admin", "pass": "secret"}'
    ```
  </Step>
</Steps>

<Note>
  The server listens on port `3000` by default. Set the `PORT` environment variable to override.
</Note>
